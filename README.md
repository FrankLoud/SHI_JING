# SHI_JING 《诗经》整理一览（极简版）


1. 《诗经》称为“诗三百”，**一共311篇**，包括305篇和另外6篇只有题目的“笙诗”

   本整理只选取了部分

   | 类型 | 原篇目数 | 所选篇目数 |
   | --- | --- | --- |
   | 风 | 160 | 85 |
   | 雅 | 105 | 34 |
   | 颂 | 40 | 12 |
   
2. 《诗经》中篇目主题历来多有争议

   因此，“主题”一栏 目前以 [程俊英, 蒋见元. 诗经注析（套装上下册）[M]. 中华书局, 1991.](https://github.com/LaiYizhou/SHI_JING/tree/master/EBook) 为主要参考
